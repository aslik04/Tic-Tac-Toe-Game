# Tic-Tac-Toe-Game

A terminal Tic-Tac-Toe game in Python with Human vs Human and Human vs Bot modes.

> Built to practice **idiomatic Python** and clean class design.

## Features
- 3 bot levels:
  - **Easy**: random
  - **Medium**: win/block + center/corners
  - **Hard**: minimax (optimal)

## Run
```bash
python3 game.py
